搭建远程会议系统(简单得集成到edx里)

Date 二 04 八月 2015 By wuwenjie Category edx. Tags edx

这一周简直就是在客串系统管理员啊，搭建各种服务还是挺有意思的

公司目前人员比较分散，挺多城市都有，可能需要召开远程会议什么的。

edx项目中似乎也有远程教学的需求，所以准备搭建一个远程会议系统

github

首先当然是去github搜索一番，键入conference。按照star降序排列我们看到：

  • WebRTC-Experiment
  • bigbluebutton
  • voicechat

筛选一番觉得bigbluebutton挺靠谱的，就试着搭建它吧

install bigbluebutton

按照官方文档就好:install

LTI相关

下一步的计划是使用LTI嵌入到edx中，这样一来在线的演讲/协同作业以及语音/视频讨论就可行了

  • learning-tools-interoperability
  • lti_component
  • installation-of-lti-module

展示一下效果(简单得集成到edx里)

目前采用的是使用js将两个系统的用户简易关联

视频模式，支持多人会议，音频设备可用

演讲模式下支持图片/pdf/word/ppt/excel 之类的东西，方便大家交流和写写画画
