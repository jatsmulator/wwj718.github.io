在微软下搭建nodejs微信后端

Date 一 16 六月 2014 By wuwenjie Category 技术. Tags wechat

教一个学妹在windows下用nodejs部署微信后台。顺便发到博客上。

前言

在此只说如何部署后台，至于注册使用微信公众号相关事宜，自行google之。
以下内容，默认你已经注册好了微信公众号。

此外所有需要的工具，我都打包在这里了,点击下载就行.之后解压它。

ngrok

在cmd中,进入ngrok所在目录，,执行ngrok.exe 80，终端将返回一个url.不要关闭这个窗口,最后一步将用到.

nodejs

点击安装node-v0.10.28-x86.msi。随意安装在哪个目录。
新版的node自带了npm，且会自动将环境变量写入PATH,所以无需配置。

npm安装依赖环境

进入到wechat目录 执行：

npm install  express
npm install weixin-robot

首先登录到腾讯微信后台， weixin:
