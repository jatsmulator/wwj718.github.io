创建你的第一个Scratch3.0 Extension

Date 三 16 一月 2019 By wuwenjie Category 少儿编程. Tags scratch

我们近期刚写完Scratch3.0技术分析系列文章，接下来准备围绕Scratch3.0编辑器写一系列文章，这一系列的文章关注如何构建自己的Extension，如何接入scratch3-adapter， 此外，我们也计划将EIM的源码开放出来。

对于你希望在自己的平台中接入scratch3-adapter, 请阅读scratch3-adapter可以支持其他编程平台吗？。

说明

文章更新于: 2019.01.16

官方的源码仓库一直在变化，我们希望即时更新到最新状态。如果你发现有任何部分不能工作，及时通知我:)

运行scratch-gui

我的开发环境为MacOS 10.13.5，确保你在本地按照了git和nodejs。使用Windows/Linux区别不大。

node -v # v10.15.0 , 推荐使用n来管理nodejs版本
npm install -g yarn
npm install -g webpack-dev-server
mkdir Scratch3
cd Scratch3
git clone https://github.com/LLK/scratch-gui

cd scratch-gui
yarn install

现在你就得到了一个可在本地运行Scratch3.0编辑器。

运行webpack-dev-server --https，打开:https://127.0.0.1:8601/

我们统一采用https来运行scratch-gui，因为以后接入scratch3-adapter需要使用https。

运行scratch-vm

cd Scratch3
git clone https://github.com/LLK/scratch-vm
cd scratch-vm
yarn install
yarn link
yarn add uglifyjs-webpack-plugin
yarn run watch

# 新开一个shell
cd scratch-gui
yarn link scratch-vm

完成yarn link scratch-vm之后，scratch-gui就会采用我们开发环境里的scratch-vm，而不是默认的scratch-vm. 这样一来我们就可以定制scratch-vm了。

scratch-vm是什么呢?

Virtual Machine used to represent, run, and maintain the state of programs for Scratch 3.0.

可见 Scratch 3.0由scratch-vm提供动力支持。你可以把它理解为运行scratch3.0积木代码的引擎/解释器。

Scratch3.0 Extension便是在scratch-vm中写的。

class Scratch3HelloBlocks {
    constructor (runtime) {
        /**
         * The runtime instantiating this block package.
         * @type {Runtime}
         */
        this.runtime = runtime;
    }


    /**
     * The key to load & store a target's pen-related state.
     * @type {string}
     */
    static get STATE_KEY () {
        return 'Scratch.helloWorld';
    }

    /**
     * @returns {object} metadata for this extension and its blocks.
     */
    getInfo () {
        return {
            id: 'helloWorld',
            name: formatMessage({
                id: 'helloWorld.categoryName',
                default: 'hello World',
                description: 'Label for the hello world extension category'
            }),
            // menuIconURI: menuIconURI,
            blockIconURI: blockIconURI,
            // showStatusButton: true,
            blocks: [
                {
                    opcode: 'say',
                    blockType: BlockType.COMMAND,
                    text: formatMessage({
                        id: 'helloWorld.say',
                        default: 'say [TEXT]',
                        description: 'say something'
                    }),
                    arguments: {
                        TEXT: {
                            type: ArgumentType.STRING,
                            defaultValue: formatMessage({
                                id: 'helloWorld.defaultTextToSay',
                                default: 'hello world',
                                description: 'default text to say.'
                            })
                        }
                    }
                }
            ],
            menus: {}
        };
    }

    say (args, util) {
        const message = args.TEXT;
        console.log(message);
        this.runtime.emit('SAY', util.target, 'say', message);
    }
}

module.exports = Scratch3HelloBlocks;
