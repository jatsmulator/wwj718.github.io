一些好用的工具

Date 日 31 七月 2016 By wuwenjie Category 工具. Tags tools

周末大淘金

unp

python包（pip install unp），能解压主流的压缩包：

  • *.7z
  • *.bz2v
  • *.tar.gz
  • *.rar
  • .zip; .egg; .whl; .jar

使用: unp myarchive.rar

mac下解压工具很难用，可以试试unp

python-prompt-toolkit

强大的交互式命令行构建框架

fabtools

方便地写fabric脚本

pluginbase

一个灵活的python插件系统

doitlive

现场的命令行演示，可以预先输入脚本，然后像放ppt一样运行他们，每个指令都会现场真是执行

moviepy

使用python做视频编辑

robobrowser

基于python的无壳浏览器

pycallgraph

将python代码的调用栈可视化

wdb

可看做web版的pdb

mac不可用

pipsi

使一般用户能安装系统级别的pip依赖

python-regex-scanner

正则提取，scanner

paperwork

文档查询 ocr （linux only）

docker安装：https://github.com/jflesch/paperwork/blob/unstable/doc/install.docker.markdown

proselint

A linter is a computer program that, like a spell checker, scans through a document and analyzes it

这是个非常酷的项目，对你的文章进行检查，并给出建议，就像对你的代码做静态分析

ncdu

找出系统大文件
