linux终端小工具使用笔记

Date 二 10 六月 2014 By wuwenjie Category 技术. Tags ubuntu

目前包括：（陆续添加）：

  • gpm
  • w3m

  • axel 参数 文件下载地址

gpm

用途

在没有图形界面的情况下使用鼠标来移动选择．

通过左键单击来选择适当的文本输入框，然后按下鼠标中键

安装

sudo apt-get install gpm

使用方法

sudo service gpm start

参考

here

w3m

用途

在命令行里上网

安装

sudo apt-get install w3m

使用方法

  • w3m URL ：打开网页
  • hjkl:移动操作类似vim
  • TAB: 转到下个超链接

更多功能，参考这里

wget

nethogs

sudo apt-get install nethogs

监控网速

使用：sudo nethogs wlan0

axel

sudo apt-get install axel

使用：axel 参数 文件下载地址，例如axel -n 10 url,开启十个线程。下载链接可以用chrometool分析
