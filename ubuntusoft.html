软件推荐

Date 四 08 八月 2013 By wuwenjie Category 技术. Tags ubuntu

以下是我常用的一些软件～

  • zim 桌面wiki，跨平台，一直用它来整理知识
  • ReTest markdown编辑器，可实时预览
  • sublime test 极其性感的文本编辑器，用它浏览代码很是赏心悦目
  • IPython IPython 是 Python 的原生交互式 shell 的增强版
  • Notebook 在web浏览器上使用ipython,极其适合做科学计算，能保存你的练习代码，让你的工作可以累加，不必为保存实验数据头疼
  • zsh The last shell you’ll ever need! 交互性极其好，懒人请点击
  • Ulipad Python ide，limodou出品,跨平台,基于wxPython，我也用它写django
  • Dia 开放源码的流程图软件
  • fcitx Linux操作系统中使用的中文输入法
  • GIMP photoshop开源替代方案
  • Docky Docky可以被视为Mac OS X的Dock的最佳克隆
  • Avant 与Docky类似的东西，感觉用着更顺手
  • Synapse 搜索系统里的任何东西，速度很快
  • 新立得 包管理工具apt的图形化前端,适合新手
  • freeming 思维导图软件
  • chrome ubuntu自带的是firefox,chrome的js调试功能让人爱不释手
  • Filezilla ftp客户端，带有sftp功能，极好用
  • Pencil 网站快速原型
  • Virtualbox 开源虚拟机软件
  • Wireshark 据说是神器，不大会用，兵器谱上排名唯一超过Metasploit的
  • evernote linux下只有everpad，印象笔记是我最重要的一个工具，没用之一
